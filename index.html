Приложение 1 · Со.Действие в стратегии СберУниверситета

Из стратегии Коммерческого блока СберУниверситета 2026–2030
#1
Флагман новой модели КБ
Бизнес-сообщества — первый из 5 флагманских проектов новой модели Коммерческого блока. Остальные: ГигаАкадемия, Универ по подписке, SberQ, Agile-модель работы.
Ответ на ключевой вызов
Вызов: «Клиент перестал покупать продукты — он хочет сопровождения». Продуктоцентричная модель устарела. Конкуренты уже работают по экосистемной модели — с подписками, сообществами и долгим жизненным путём.
Жизненный путь клиента — мы на этапах 4–5
1. Осознание 2. Запрос 3. Обучение 4. Развитие 5. Амбассадор
Со.Действие — это этап регулярного развития (74 касания/год, LTV вместо сделки) и амбассадорства (клиент приводит новых, круг замыкается). Кампус — из места обучения в среду развития.
Финмодель СУ-2030
Без КИБ-эффекта: IRR 0.2%, окупаемость 47 лет
С КИБ: IRR 15%, окупаемость 10.9 лет
Синергия с кампусом
35 тыс. участников сообществ — новый поток B2B-клиентов для кампуса. Сообщества не заменяют, а заполняют актив.

Приложение 3 · Гига Академия

Оператор популяризации AI-компетенций. Механизм роста экосистемы, где каждый обученный генерирует доход через потребление токенов.
Сообщество — среда на каждом этапе
Сообщество не только в конце воронки. На каждом этапе оно повышает конверсию: обучение → покупка → потребление. Среда доверия = дополнительная ценность на всех уровнях.
